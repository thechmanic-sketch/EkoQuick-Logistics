Driver Settings

These are enforced for real: registration can be closed, new drivers can skip manual approval, and Dispatch/auto-assign will only match drivers within these limits.

Security

This app currently has one authentication role ("admin"). Staff Role below is an additional layer on top of that — it's enforced client-side (e.g. Commissions already checks for Super Admin) but not yet at the database level for every module, since that would require rewriting Row Level Security policies across every table. Treat it as real but not yet airtight against a technically malicious admin.

Appearance

This app uses one deliberately-chosen dark "waybill" theme across every page — a Light Mode / System Theme variant was previously discussed and explicitly deferred (would require re-theming every page).
